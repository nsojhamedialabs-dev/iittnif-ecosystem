# ui

The six reusable modules from PDR Section 08, plus components added for
the beyond-brief commitments. One component, re-themed per page via the
design-system tokens — not eight one-off builds.

- `Hero/` — full-bleed statement + 2 CTAs
- `Pillars/` — 3–6 card grid, hover reveal
- `Pipeline/` — animated horizontal flow (Research → Impact, re-scoped per page)
- `Portfolio/` — filterable card grid
- `Stories/` — photo + narrative + outcome
- `EngageCTA/` — stakeholder card router (routes into VIKAS)
- `Glossary/` — hover-tooltip inline glossary (PDR beyond-brief: GNSS, GeoAI, etc.)
- `LanguageToggle/` — EN/HI toggle, default-on bilingual layer
- `ImpactCounter/` — live counter component, consumes cms-client — fixes Dronagiri's zero-stat problem
- `Map/` — interactive India map (Mapbox/Leaflet per PDR Section 07), used in Dronagiri /pilots and iittnif-web SPIN Labs section
